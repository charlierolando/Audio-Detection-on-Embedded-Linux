Audio Detection on Embedded Linux
• Detect audio on the embedded Linux audio jack, whether there is sound or no sound
• Running well in Orange Pi 3B which has the Debian variant of Linux installed
• pip install sounddevice
• run audio_detection.py https://github.com/charlierolando/Audio-Detection-on-Embedded-Linux/blob/main/source/audio_detection.py
• 'hw:0,0' you can replace it with the appropriate name or device identifier, in this case identifying the mic connector on the audio jack
